# NuwaGradle
gradle plugin wrote for [Nuwa](https://github.com/jasonross/Nuwa)


##Useage

```groovy
 repositories {
        jcenter()
        maven { url 'https://dl.bintray.com/dodola/maven' }
    }

    dependencies {
         classpath 'cn.jiajixin.nuwa:gradle:1.2.3'
     }
```
